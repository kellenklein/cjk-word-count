# cjk-word-count
A simple command-line tool written in Rust to count the number of graphemes/letters (non-punctuation, non-special-characters, etc) in a single block of text.

## How to use
Paste in the text you want to count. Text input is separated by line break (the enter key) so in order to have all of your text in a single "count" you should remove any enters/line breaks from your text.
